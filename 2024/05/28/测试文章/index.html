Node.js

  • 什么是Node.js

    • Node.js 是一个开源、跨平台的运行时 (Runtime)环境，用于在服务器端运行 JavaScript 代码。本身不是框架或引擎，但它使用 V8 引擎来执行 JavaScript 代码，并且有许多基于它构建的框架来帮助开发者更高效地构建应用程序。

  • 基于Node.js的框架

    • Node.js 是一个 JavaScript 运行时，基于 V8 引擎，允许在服务器端运行 JavaScript 代码。
    • Express 是一个基于 Node.js 的 Web 应用框架，提供了简洁的 API 和工具集，用于构建 Web 服务器和处理 HTTP 请求。
      • Koa：由 Express 的原始开发者创建的下一个生成的 Web 框架，旨在更小、更富表现力、更健壮。
      • NestJS：一个用于构建高效、可扩展的 Node.js 服务端应用的框架，具有良好的模块化和依赖注入。

  • 特点和主要功能

    • 单线程事件驱动架构
    • 非阻塞 I/O
    • 附带 npm
    • 跨平台
    • 适用于实时应用
    • 丰富的生态系统

  • 主要应用场景

    • Web 服务器：利用 Express.js 等框架，Node.js 可以快速构建高性能的 Web 服务器。
    • API 服务器：Node.js 常用于构建 RESTful API 和 GraphQL API，支持前后端分离开发。
    • 实时应用：如实时聊天、实时协作工具、在线游戏等。
    • 流媒体应用：由于其非阻塞 I/O 特性，Node.js 适用于处理流媒体数据传输。
    • 工具和自动化脚本：Node.js 可以用来编写各种开发工具、构建系统和自动化脚本。
